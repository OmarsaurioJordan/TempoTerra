# TempoTerra

Requiere Godot 4

Iniciado en 2022 (GMS) y retomando en 2026 (Godot)

Juego 2D isométrico, de disparos, se puede mover en el tiempo, por diferentes épocas de la humanidad, llevando y trayendo NPCs entre las épocas, se propone agregar narrativas educativas o de aventura
